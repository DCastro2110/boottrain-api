# Subtask 3: Create User repository

## Description

Create `src/db/user-repository.ts` implementing IUserRepository with Prisma.

## Details

1. Create `src/db/user-repository.ts` file
2. Import necessary Prisma client and types
3. Implement `UserRepository` class with `IUserRepository` interface
4. Implement `findById` method:
   - Query Prisma for user by ID
   - Return null if not found
   - Return formatted user data
5. Implement `update` method:
   - Accept optional transaction parameter (tx) for consistency with other repositories
   - Update only provided fields using Prisma update
   - Return `{ id: string }`

## Acceptance Criteria

- [ ] Implements IUserRepository interface
- [ ] findById returns user data or null
- [ ] update method supports partial updates (all fields optional)
- [ ] Supports transaction parameter (tx) like other repositories
- [ ] Uses Prisma client from `src/lib/db.ts`
- [ ] The code compiles without errors
