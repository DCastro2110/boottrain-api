# Subtask 5: Create UpdateUserDataUseCase

## Description

Create `src/usecases/update-user-data-use-case.ts`.

## Details

1. Create `src/usecases/update-user-data-use-case.ts` file
2. Define `IUpdateUserDataUseCase` interface
3. Define `InputDTO` with `userId` and all optional biometric fields:
   - height?: number
   - weight?: number
   - age?: number
   - fitnessLevel?: "beginner" | "intermediate" | "advanced"
   - bodyFatPercentage?: number
4. Define `OutputDTO` with `userId: string`
5. Create `UpdateUserDataUseCase` class:
   - Constructor accepts `IUserRepository` dependency
   - `execute(input: InputDTO): Promise<OutputDTO>` method
   - Validate that at least one field is provided
   - Call `repository.findById` to check user exists
   - Call `repository.update` with provided fields
   - Return `{ userId: string }`

## Acceptance Criteria

- [ ] IUpdateUserDataUseCase interface is defined and exported
- [ ] InputDTO contains userId and all optional biometric fields
- [ ] OutputDTO contains userId
- [ ] Calls repository.update with provided fields
- [ ] Throws NotFoundError if user doesn't exist
- [ ] Has explicit return type on class and execute method
- [ ] The code compiles without errors
