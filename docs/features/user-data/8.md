# Subtask 8: Run DB migration

## Description

Run `prisma migrate dev` to apply schema changes.

## Details

1. Ensure Docker Postgres is running: `docker compose up -d postgres`
2. Run `pnpm prisma migrate dev` to create and apply migration
3. Provide a migration name like "add_user_biometric_fields"
4. Run `pnpm prisma generate` to regenerate Prisma client

## Acceptance Criteria

- [ ] Migration is created successfully
- [ ] Database schema is updated
- [ ] Prisma client is regenerated without errors
