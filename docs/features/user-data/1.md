# Subtask 1: Add fields to User model in Prisma schema

## Description

Add height, weight, age, fitnessLevel (enum), and bodyFatPercentage fields to the User model. Create the fitnessLevel enum.

## Details

1. Read the current `prisma/schema.prisma` to understand the existing User model structure
2. Add a new enum `FitnessLevel` with values: "beginner", "intermediate", "advanced"
3. Add the following fields to the User model:
   - `height`: Int? (centimeters) - nullable
   - `weight`: Int? (grams) - nullable
   - `age`: Int? (years) - nullable
   - `fitnessLevel`: FitnessLevel? - nullable
   - `bodyFatPercentage`: Int? (0-100) - nullable

## Acceptance Criteria

- [ ] A new enum "FitnessLevel" exists with values: "beginner", "intermediate", "advanced"
- [ ] The User model has all five new fields with correct types and nullability
- [ ] The code compiles without errors after running `prisma generate`
- [ ] A DB migration is created with `prisma migrate dev`
