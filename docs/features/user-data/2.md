# Subtask 2: Create User domain interface

## Description

Create `src/domain/user.ts` with IUserRepository interface and IUserData type.

## Details

1. Create `src/domain/user.ts` file
2. Define `IUserData` interface with all user biometric fields:
   - id: string
   - email: string
   - name: string | null
   - height: number | null
   - weight: number | null
   - age: number | null
   - fitnessLevel: "beginner" | "intermediate" | "advanced" | null
   - bodyFatPercentage: number | null
   - image: string | null
3. Define `IUserRepository` interface with methods:
   - `findById(id: string): Promise<IUserData | null>`
   - `update(id: string, data: UpdateUserDataInput): Promise<{ id: string }>`
4. Define `UpdateUserDataInput` type with optional fields for height, weight, age, fitnessLevel, bodyFatPercentage

## Acceptance Criteria

- [ ] IUserRepository interface is exported with findById and update methods
- [ ] IUserData type is exported with all biometric fields
- [ ] Follows naming conventions (I prefix for interfaces)
- [ ] Uses proper TypeScript with explicit return types
- [ ] The code compiles without errors
