# Subtask 4: Create GetUserDataUseCase

## Description

Create `src/usecases/get-user-data-use-case.ts`.

## Details

1. Create `src/usecases/get-user-data-use-case.ts` file
2. Define `IGetUserDataUseCase` interface
3. Define `InputDTO` with `userId: string`
4. Define `OutputDTO` with all user biometric fields
5. Create `GetUserDataUseCase` class:
   - Constructor accepts `IUserRepository` dependency
   - `execute(input: InputDTO): Promise<OutputDTO>` method
   - Call `repository.findById(userId)`
   - Throw `NotFoundError` if user doesn't exist
   - Return formatted user data

## Acceptance Criteria

- [ ] IGetUserDataUseCase interface is defined and exported
- [ ] InputDTO contains userId
- [ ] OutputDTO contains all user biometric fields
- [ ] Returns user data by calling repository.findById
- [ ] Throws NotFoundError if user doesn't exist
- [ ] Has explicit return type on class and execute method
- [ ] The code compiles without errors
