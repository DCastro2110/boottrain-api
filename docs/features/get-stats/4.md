# Subtask 4: Register Stats Route in index.ts

## Description

Add the stats route to the Fastify app in `src/index.ts`.

## Details

Register the stats route in the Fastify application:

- Import `statsRoutes` from `./routes/stats.route.js`
- Register with `app.register(statsRoutes)`

## Acceptance Criteria

- `stats.route.js` is imported with `.js` extension
- Route is registered with the app
- Typecheck passes (`pnpm tsc --noEmit`)
- Lint passes (`pnpm eslint .`)
