# Subtask 5: Register Stats Route in index.ts

## Description

Add the stats route to the Fastify application in `src/index.ts`.

## Details

The route registration should:

- Import `statsRoutes` from `./routes/stats.route.js`
- Register the route with the app: `app.register(statsRoutes)`

## Changes to src/index.ts

```typescript
import { statsRoutes } from "./routes/stats.route.js";

// In the app setup function, after other route registrations:
app.register(statsRoutes);
```
