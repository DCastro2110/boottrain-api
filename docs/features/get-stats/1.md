# Subtask 1: Create Domain Interface for Stats Repository

## Description

Create `src/domain/stats.ts` with the `IStatsRepository` interface that defines the contract for fetching workout sessions by user and date range.

## Details

The interface should define a method to fetch workout sessions given:

- `userId`: The authenticated user's ID
- `startDate`: Start of date range (inclusive)
- `endDate`: End of date range (inclusive)
- Optional `tx`: Transaction client for atomic operations

## Interface Design

```typescript
import type { tx } from "../types/utils.js";

export interface IStatsRepository {
  findByUserIdAndDateRange(
    userId: string,
    startDate: Date,
    endDate: Date,
    tx?: tx,
  ): Promise<IWorkoutSessionForStats[]>;
}

export interface IWorkoutSessionForStats {
  id: string;
  startedAt: Date;
  completedAt: Date | null;
}
```

## Acceptance Criteria

- File exists at `src/domain/stats.ts`
- Exports `IStatsRepository` interface
- Exports `IWorkoutSessionForStats` interface
- Interface method signatures match expected usage in use case
- Uses ESM imports with `.js` extension
- Follows `I` prefix naming convention for interfaces
