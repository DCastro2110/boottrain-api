# 5. Implement today workout day payload composition

## Description

Resolve today’s `WorkoutDay` from active plan and include required fields plus completion status for today.

## Acceptance Criteria

- `todayWorkoutDay` includes: `date`, `name`, `estimatedDurationInSeconds`, `numberOfExercises`, `coverImageUrl`, `isCompleted`.
- `date` is formatted `YYYY-MM-DD`.
- `numberOfExercises` reflects count of related workout exercises.
- If no workout day exists for today (or no active plan), `todayWorkoutDay` is `null`.
