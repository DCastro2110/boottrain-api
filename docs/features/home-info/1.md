# 1. Define feature contract and endpoint scaffolding

## Description

Create `home-info` route skeleton with authentication and Zod response schemas, mirroring `workout-plan` route style. Wire it in `src/index.ts` with route prefix.

## Acceptance Criteria

- The server registers a new route module `src/routes/home-info.route.ts`.
- Route is exposed under prefix `home-info` and method `GET /`.
- Route returns `401` using existing `ErrorSchema` when session is missing.
- Route response schema defines `weekConsistency`, `fireSequence`, and nullable `todayWorkoutDay`.
- Imports follow ESM `.js` and project lint conventions.
