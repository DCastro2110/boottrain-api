# 8. Verification and quality checks

## Description

Validate implementation with project-standard static checks and scenario-based manual API checks.

## Acceptance Criteria

- `pnpm tsc --noEmit` passes.
- `pnpm eslint .` passes.
- Manual checks confirm expected outputs for:
  - no sessions this week,
  - some completed + some missed days,
  - no active plan,
  - completed today vs not completed today.
- Route is visible in Swagger/Scalar via existing docs setup.
