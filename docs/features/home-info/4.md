# 4. Implement week consistency and fire sequence rules in use case

## Description

Build deterministic date/week mapping (Sunday→Saturday), assign daily statuses, and compute fire sequence according to agreed rule.

## Acceptance Criteria

- `weekConsistency` always has exactly 7 ordered entries from sunday to saturday.
- Status values are strictly one of `completed | not_completed | missed`.
- Fire sequence is an integer computed from consecutive completed days (backward from today in current week).
- Behavior is deterministic for empty sessions, partial sessions, and past/future days.
