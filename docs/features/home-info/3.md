# 3. Implement Prisma HomeInfoRepository queries

## Description

Create `src/repositories/home-info-repository.ts` implementing `IHomeInfoRepository` with efficient reads for active plan, week sessions, and today workout day + exercise count.

## Acceptance Criteria

- Repository class compiles and implements all interface methods.
- Data retrieval is scoped by authenticated `userId`.
- Queries include sufficient data to compute statuses, streak, and today payload without N+1 loops.
- Repository returns normalized raw data for use case computation.
