# Task 6: Create src/db/ and move repository implementations

## Description

Create the `src/db/` folder and move all repository implementation files from `src/repositories/` to `src/db/`.

## Actions

1. Create the `src/db/` directory
2. Move `src/repositories/workout-plan-repository.ts` to `src/db/workout-plan-repository.ts`
